ABOUT THIS PROJECT

Welcome to GAMER GRID, the eccentric sister-project of Media Masterlist! Media Masterlist is designed to be a simple yet sleek table for tracking your favourite media: books, music, movies, and more. When I shared my Media Masterlist project, I got some great feedback that I should use a system like the IGBD API to pull covers for each entry to make things more visually interesting.

This API is specific to video games, so I decided to create a spinoff project focused entirely on games rather than all kinds of media. This also gave me the opportunity to experiment with some fun cyberpunk styling to fit the theme!


HOW TO USE

Just type in a game title in the first column, and a cover for the game should appear. Then, you can give the game a rating, add additional info like the genre or platform, and even write a review! Click the save button to save your progress for the next time you visit the page, or click export to export your list as a CSV.

If you don't like the cover that's pulled, you can copy an image and paste it in manually. Here's an example of how the table can look when filled out, using my own personal Top 10 Games List.
