WHAT IS THIS?

Welcome to GAMER GRID, the eccentric sister-project of Media Masterlist! Media Masterlist is designed to be a simple, sleek and sophisticated table for tracking your favourite media: that can include books, music, television shows, and more. I wanted to make a more technical project with a similar concept using an API. Creating a playful game-based project seemed like a fun opportunity to try something new and implement some over-the-top retro cyberpunk vaporwave styling!


WHAT INSPIRED THIS PROJECT?

When I shared my Media Masterlist project, I got feedback that I should try implementing IGBD API so that when someone types in a game title, a game cover would generate beside the entry. I loved the idea, but I wanted to keep Media Masterlist flexible and minimalistic, without restricting users to just listing games. That's when I realized that this would be a fun opportunity to make a new project focused on video games with some maximalist styling! I wanted to have a lot of fun with the design, which was partially inspired by the fun retro styling of the DO IT Hackathon web page.
